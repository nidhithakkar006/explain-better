# Design & Wireframes

This folder contains low-fidelity wireframes for **ExplainBetter**.

The wireframes were created before implementation to:
- Define the end-to-end user flow
- Reduce ambiguity during development
- Validate UX decisions early

## MVP Flow
1. Landing / Start
2. Concept Selection
3. Mode Selection
4. Explanation Input
5. Feedback Summary
6. Keyword-Guided Retry
7. Progress & History
